===================================================================
                    Welcome to FrameBridge
              Universal Graphics Runtime Installer
===================================================================

FrameBridge is an open-source platform built around the OptiScaler
Graphics Runtime. It simplifies the installation of modern graphics
enhancements by automatically configuring OptiScaler together with
Nukem9's DLSSG-to-FSR3 Frame Generation mod.

The goal of FrameBridge is to provide a clean, stable, and unified
experience across NVIDIA, AMD, and Intel GPUs while minimizing
manual setup and compatibility issues.

-------------------------------------------------------------------
                     INSTALLATION INSTRUCTIONS
-------------------------------------------------------------------

TARGET DIRECTORY

Choose the folder containing the game's executable (.exe).

Examples:
    Game\
    Game\bin\
    Game\Binaries\Win64\

FrameBridge must be installed in the same directory as the game
executable.

-------------------------------------------------------------------

EXISTING FILES

If the installer detects an existing proxy DLL (dxgi.dll,
version.dll, d3d12.dll, etc.), it may belong to another mod or a
custom compatibility fix.

Unless you know exactly what the file is, do NOT overwrite it.

If you are unsure, cancel the installation, investigate the existing
mod first, then reinstall using a different injection method if
necessary.

-------------------------------------------------------------------

MANUAL INSTALLATION

You may also install FrameBridge into an empty folder.

This allows you to inspect the generated files before copying them
manually into the game's directory.

-------------------------------------------------------------------

INJECTION METHOD

• DXGI (Recommended)
  Native OptiScaler runtime.
  Best choice for most DirectX 12 games and the default installation.

• Version.dll (Advanced)
  Legacy compatibility option.
  Intended only for games that specifically require Version proxying.

• D3D12.dll (Advanced)
  Only use if recommended for your game or by project
  documentation.

-------------------------------------------------------------------
                        IN-GAME USAGE
-------------------------------------------------------------------

1. Launch the game.

2. Open the game's graphics settings.

3. Select DLSS if the game provides the option.

   FrameBridge and OptiScaler automatically redirect the requested
   upscaling technology to the most appropriate backend supported by
   your hardware.

4. If supported by the game, enable Frame Generation.

5. Press the INSERT (INS) key at any time to open the OptiScaler
   overlay.

From the overlay you can:

 • Select the rendering backend
 • Configure Frame Generation
 • Adjust image sharpening
 • Enable compatibility options
 • View runtime information

-------------------------------------------------------------------
                           IMPORTANT
-------------------------------------------------------------------

MULTIPLAYER GAMES

Do NOT use FrameBridge with games protected by anti-cheat systems
(Easy Anti-Cheat, BattlEye, Vanguard, Ricochet, etc.).

DLL injection may violate the game's Terms of Service and can result
in account penalties.

-------------------------------------------------------------------

ANTIVIRUS WARNINGS

Because FrameBridge installs proxy DLLs and performs graphics API
hooking, some antivirus software (including Microsoft Defender) may
incorrectly flag the files.

These detections are common for graphics wrappers and modding tools.
